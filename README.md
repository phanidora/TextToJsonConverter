# TextToJsonConverter

Java project that reads the attached file (data.txt) and parses it into a text file. The text file should be equal to the below json data structure:
{
  'host': 'test.com',
  'server_id': 55331,
  'cost': 2.56,
  'user': 'user',
  'verbose': true,
  'test_mode': true,
  'debug_mode': false,
  'log_file_path': '/tmp/logfile.log',
  'send_notifications': true
}
